Minimal Collection

A component library by @kenliten

Intro

Minimal have a default color palette. You can (and should) changes this values for yours.

These are the default theme color values: 

--disabled-bg-color: #F5F5F5;
--default-bg-color: #EEEEEE;
--default-bg-hover-color: #E0E0E0;
--default-txt-color: #636363;
--default-txt-hover-color: #212121;
--primary-bg-color: #BF360C;
--primary-bg-hover-color: #DD2C00;
--primary-txt-color: #F5F5F5;
--primary-txt-hover-color: #FFFFFF;
--secondary-bg-color: #424242;
--secondary-bg-hover-color: #000000;
--secondary-txt-color: #F5F5F5;
--secondary-txt-hover-color: #FFFFFF;
