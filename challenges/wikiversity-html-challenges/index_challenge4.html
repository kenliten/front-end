Lists, lists and more lists

There are 3 types of lists:

Definition lists
Which show terms and definitions
Unordered lists
Which show bullets
Ordered lists
Which show numbers or letters

Above is an example of a definition list.

Unordered lists can have a type attribute which can be applied to the UL or the LI tag.
The three values allowed are:

  • type = "square"
  • type = "disc"
  • type = "circle"

Ordered lists can have a type attribute too. Used along with a start attribute they can show:

  1. Lower case letters
  2. Upper case letters
  3. Lower case roman numerals
  4. Upper case roman numerals
  5. Numbers
